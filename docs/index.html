🚀 快速开始

Docker 部署（推荐）
# 克隆项目
git clone https://github.com/FASUZO/PartShelf-V2.git
cd PartShelf-V2

# 启动服务
docker-compose up -d

# 访问应用
# http://localhost:8000
本地开发
# 安装依赖
pip install -r requirements.txt

# 启动服务
uvicorn app.main:app --reload

# 访问应用
# http://localhost:8000
